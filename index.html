WBB Plugin - BBCode um USK Altersbewertungen in den Post einzufügen / WBB Plugin - BBCode to display USK Ratings in your Post

Download Download View BBCode USK on GitHub

Fügt die USK Bewertungen ein Der BBCode fügt die USK Bewertungen für Spiele ein.

Die Syntax sieht wie folgt aus: [usk='$alter'(,'$größe')][/usk]

$alter lässt eine der fünf Altersangaben zu $größe nimmt Werte von 1-99
